# Tools

Validators, generators, and verification tools for the `sdkwork-settings` repository.

Authority: `../sdkwork-specs/CODE_STYLE_SPEC.md`, `../sdkwork-specs/TEST_SPEC.md`.

## Rules

- Repository-specific validators and generators live here.
- Cross-repository standard tools live under `../sdkwork-specs/tools/` and are invoked by relative path.
- Tools must produce evidence bundles consumable by quality gates per `QUALITY_GATE_SPEC.md`.
